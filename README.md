# Vityarthi-25BCE10593
Supermarket / Warehouse Indexer
This project is useful in indexing various items and alerting you if an item is low in stock and the sell cost and buy cost of the price dont result in a profitable outcome.

To Install/Open the program :
1) Install the " supermarket_index.py " file from this github page.
2) Open the "supermarket_index.py" file using your prefered IDE
3) Run the program and utilize it as a basic indexing program.

To Use the Program:
-After running the program you may choose one of 5 options to complete your task
1) 1 Add an item to the Index
2) 2 List items in the Index
3) 3 Change the stock available for an existing item
4) 4 Remove an item from the list
5) 5 Quit or exit the program.
