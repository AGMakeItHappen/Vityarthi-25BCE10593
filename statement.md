Problem Statement : Often when maintaining inventories in warehouses and supermarkets there are numerous errors made causing miscalculations and eventual losses for these organizations.

Scope of the project : An attempt to create an index to help these organizations/companies index their inventories to better maintain and keep track of items that they have.

Target Users : Target users are Warehouses, Companies with large and variable inventories, Supermarkets and so on.

High Level features : usage of modular programming and the logic behind error handling.
